Connecting to Methodist oneChart

Windows PC

In order to access Methodist oneChart, your PC or laptop hardware must meet the minimum specifications and have the correct version of the Citrix Web Client. Please click the link below for instructions on how to get started with Methodist oneChart:

Download the oneChart Getting Started Technical Guide

Citrix Client

You will need to download and install the Citrix Web Client to access Methodist oneChart. Instructions for installing the Citrix client are in the oneChart Getting Started Technical Guide. Once you have read the instructions, please use the link below to install the Citrix Web client.

Download the Citrix Web Client (PC/Windows)

iOS / Android Instructions

In order to connect iOS devices to access the Guest Network and oneChart, you will need to have an iTunes account setup, Apple iOS 4.2 or higher installed, and the Citrix Receiver App installed. Please use the link below for detailed instructions on connecting your iOS device to the Guest Network.

Download the iPad/iPhone/iOS instructions

Mac Users

Download the oneChart Getting Started Technical Guide

Methodist oneChart is also accessible on Mac OS X, version 12.9.1 computers. Please link below for instructions on how to download and install the Citrix Web Client for your Mac.

Download the Citrix Web Client (Mac)

Note: Though access to oneChart is possible from a Mac, Methodist does not provide support for non-Windows devices.
